# 部落格文章撰寫風格指南

## 禁止使用 Emoji Icon

撰寫部落格文章時，**請勿在文章內容中使用任何 Emoji icon**（例如 🧠❤️⚖️🚗⚠️📋🏥💡🎯✅😴❌💊🔧📌🔬 等）。

### 原因
- 減少 AI 生成感，提高人類撰寫文章的感受
- 維持醫學專業文章的正式性與可信度
- 讓讀者能感受到這是經人類作者審核過的文章

### 適用範圍
- 所有流程圖中的標題與內容
- 所有表格中的欄位文字
- 所有提示框、注意事項區塊
- 所有章節標題與內文
- Take Home Message 等總結區段

### 替代做法
- 直接使用純文字標題，不加任何裝飾性符號
- 以粗體、色彩、邊框等 CSS 樣式來區分重點，而非依賴 Emoji
